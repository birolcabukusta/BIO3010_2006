# Data analysis workshop for BIO3010
## Bachelor's degree 
## Maastricht University

**Learning Goal:**
* Writing a simple omics data analysis script using R and RStudio to understand the fundamentals of omics data analysis.
* Understand that all omics data is different and needs to be handled uniquely, and there is no silver bullet to handle all types of data
* 

**Requirements:**
* A computer with R and RStudio installed.
* Internet connection
